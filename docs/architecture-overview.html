🧠 智能咨询平台
"让建议可自动生成"
📋 甲方得到
  • 专业投资分析报告 — 一键生成 Word / PDF / PPT
  • 智能问答 — 用自然语言问数据、问风险、问建议
  • 风险仪表盘 — 综合评分 + 关键指标卡片 + 告警
  • 多项目组合对比 — 横向比较不同投资项目
  • 客户端口 — 投资方可独立查看报告和仪表盘
⚙️ 开发构建
  • 报告引擎 — 模板驱动 + 图表自动生成 Word/PDF/PPT
  • LLM 问答引擎 — RAG 检索增强 + 多轮对话管理
  • 叙述生成器 — 分析结果 → 自然语言解读
  • 可视化仪表盘 — React + ECharts 交互图表
  • 多项目服务 — 组合管理 + 跨项目指标对比 API
↑ 风险量化结果 驱动 报告与建议生成
2
📊 风险分析平台
"让风险可量化"
📋 甲方得到
  • 悲观 / 基准 / 乐观情景对比 — 自动计算三套指标
  • 敏感性矩阵 + 龙卷风图 — 哪个参数影响最大？
  • 蒙特卡罗概率模拟 — IRR / NPV 的概率分布
  • 盈亏平衡分析 — 找到投资临界点
  • 历史对比 — 不同时间点的分析结果可追溯对比
⚙️ 开发构建
  • 情景 / 敏感性 / MC 分析引擎 — 核心算法封装为服务
  • 并行计算框架 — Celery + Redis 异步任务队列 1000次MC<60s
  • 派生指标计算 — IRR / NPV / DSCR / 回收期
  • 风险指标引擎 — VaR / 压力测试 / 信用风险评估
  • 结果持久化 — PostgreSQL 存储分析历史 + 版本对比
↑ 结构化数据 驱动 多维风险分析
1
🏗️ 数据基础平台
"让数据可计算"
📋 甲方得到
  • Excel 自动解析 — 上传模型文件，自动识别表结构和指标
  • 三层知识图谱浏览 — Sheet → 表 → 指标 → 单元格逐层下钻
  • 修改参数 → 即时重算 — 改一个电价，全模型自动更新
  • 快照对比 — 保存不同方案，任意两版差异一目了然
  • Excel 导出 — 重算结果导出为标准 Excel 文件
⚙️ 开发构建
  • 解析引擎 — openpyxl 双模式 + 公式词法分析 + 依赖提取
  • 三层图谱引擎 — Cell → Indicator → Table Neo4j
  • 增量重算引擎 — Kahn 拓扑排序 + 快速路径优化 42%命中
  • API 平台 — FastAPI + JWT 认证 + OpenAPI 文档
  • 数据存储 — PostgreSQL + Neo4j + Redis + MinIO
